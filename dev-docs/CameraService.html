(static) this.uploadPicture() → {HttpPromise}

Uploads a picture, given the picture's file path, to the server for a normal upload, or uploads the compressed image using JIC if using the smartUpload setting. Checks if the device is online, that the 10-minute anti-spam period is up, and that the provided filePath is of a correct type before trying to upload to the server.
